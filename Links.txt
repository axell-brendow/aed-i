    IMPORTANTE:

          Os arquivos
 
             IO.jar 
             e 
             jkarel.jar 

          deverão ser copiados 
          para uma pasta (extra) a ser criada
          onde o Java for instalado (JAVAHOME)

          JAVAHOME\extra

          Para versões anteriores deverão ser usadas
          as pastas

          JAVAHOME\jdk\jre\lib\ext e
          JAVAHOME\jre\lib\ext
          (ou similares, se houver).

          É conveniente a definição de uma variável de ambiente
          no sistema operacional para facilitar usos futuros:

          CLASSPATH=JAVAHOME\extra\IO.jar;JAVAHOME\jdk\jkarel;.\
